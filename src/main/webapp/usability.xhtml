About the System - Project 3: JSF + Hibernate + Primefaces

This system registers users on a form with validations and offers the option to upload a photo. It is the only project in the portfolio that does not have login control. Once saved, the register will be added to a list with other registers, who have already been saved in the system. From there, there is the option to add a limited number of telephone numbers and emails for each register. It is possible to search according to a user's first or last name, download images and generate a PDF with search results. An age comparison chart is automatically generated with each new record saved.

Tolls - Technical informations: Eclipse IDE, Postgresql, EC2 Amanzon Tecnology (deploy on cloud Server). Java Server Faces, Hibernate, CDI, Prime Faces, Javascript.
